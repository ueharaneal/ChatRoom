NU ChatRoom

About

Summary

NU ChatRoom is an interactive, user-friendly web application designed for seamless communication in real-time. Built on the robust Flask framework and styled with Tailwind CSS for a sleek, responsive user interface, this application offers a dynamic environment for users to create and join chat rooms.

Quick Start

Creating a room.

  1. 1. Find the "Create a ChatRoom" form.
  2. 2. Input a name and press the "Create a Room" button.
  3. 3. Your new room will be created and the room number will be displayed on the top for other users to join
  4. 4. Chat Away!.

Joining a room.

  1. 1. Find the "Enter a ChatRoom" form.
  2. 2. Input a username and the desired room number.
  3. 3. Press the "Create a Room" button.
  4. 4. You will now join that room and the room number will be displayed on the top for other users to join
  5. 5. Chat Away!.

Key Features:

Room Creation and Joining:

  • Users can easily create new chat rooms, each assigned a unique room code.
  • Joining a room is as simple as entering the room code, enabling quick access to different chat spaces.

Real-Time Communication:

  • Experience live, real-time interactions with others in the chat room.
  • The application ensures messages are instantly delivered, fostering an engaging conversation experience.

Technology Used:

  • Backend: Flask (Python-based web framework)
  • Frontend: Tailwind CSS for styling
  • Socket IO: Real time interactions
